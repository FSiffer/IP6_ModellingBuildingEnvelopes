IIT22: Modelling Building Envelopes with the Envelop Sketchup Extension

Summary

Envelop is a Sketchup extension for quickly measuring building surface areas by material and facing.

Keywords

3D Modelling, Building Envelopes, Trimble Sketchup, Ruby, Javascript, HTML, CSS, Usability

Goals

The goal of this project is to enalbe the customer, Planfabrik GmbH, to model a building in 3d, based on 2d plans. This model must then be able to output its surface area totals, grouped by material such as window or door and by facing such as north. The entire process should take about 15 minutes.

Previous Workflow

Before this project, the buildings had to be fully modeled using Plancal Nova. This included detailed information regarding windows, doors and materials. Not only did Plancal Nova require much more information than was necessary to perform the required measurements, entering this information was also very slow and burdensome, requiring the navigation of many layers of menus and having very little support for a visual and interactive modeling workflow.
Having to enter more information than would be necessary and having to use work with Plancal Novas user interface meant that even for very experienced users, going from two dimensional building plans to calculated building envelope surface area totals, required about an hour to an hour and a half of concentrated working.

Results



The Envelop extension for Sketchup was developed as part of this project. It solves the customers needs, by allowing him to make the desired measurements about four to five times faster than previously - while still maintaining the required precision. Such a measurement can be seen in full in the video at the top of the page.
The massive times savings where realised by developing a simple and flexible visual workflow. Plans are select based on a visual preview and positioned, as well as scaled, using two clicks. Outlines can be traced by clicking the corners. Volumes are created by simply dragging on an shape. Materials are applied by clicking surfaces. Envelop also ensures the user scales and orientates the model correctly before providing results. Moreover, in order to allow novice users to learn how to use the extension, without having to invest time into reading the manual, there is a quick guide including step by step instructions as well as short demonstrative videos.

Envelop can be downloaded with the following link: Envelop.rbz.

It's source code, including the project report, manual and testing artifacts haven been made public as a open source repository under the following link: IP6_ModellingBuildingEnvelopes.
